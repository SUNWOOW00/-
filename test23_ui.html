HTML의 특징
  • HTML은 웹 문서의 표준형식이다.
  • HTML문서는 공백이나 줄바꿈을 인식하지않는다.
  • 대소문자를 구분하지 않는다.
  • 시작 태그와 종료 태그로 되어있다.
HTML문서의 구성요소
  • 요소 : 시작 태그와 종료 태그로 이루어진 모든 명령어 (type = circle)
  • 태그 : 요소(element)의 일부로 시작태그와 종료태그 두 종류가 있다
  • 속성 : 요소의 시작 태그 안에서 사용되는 것으로 구체화된 명령어 체계
  • 변수 : 속성에 관련 된 값
HTML 문자 관련 태그
  • H 태그: Heading(문서 제목)의 의미로, h1~h6(type = square)
  • 글꼴(front) 태그 : 속성 정보를 이용하여 size, color, face(글자체)
  • 글자 모양 변경 태그 : b, i, u, tt, dfn, cite, code, sub, sup, big
  • 특수문자 표시 : 공백문자, 특수문자 >, <, &, "
  • <br> : 줄 바꿈을 나타내는 태그
